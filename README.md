# OSSU UI

**Open Source Society University client-side application**

# Installation

1. Clone the repository

  ```bash
  $ git clone git@github.com:open-source-society/ossu-ui.git
  $ cd ossu-ui
  ```

1. Load the dependencies (via NPM)
  ```bash
  $ npm install
  ```
1. Start Dev Server
  ```bash
  $ npm start
  ```
